## Installation

Install pumpkin-webshop locally with npm

```bash
# Clone this repository
$ git clone https://github.com/ellinorjohansson/pumpkin-webshop

# Go into the repository
$ cd pumpkin-webshop

# Install dependencies
$ npm install

# Run app
$ npm run dev
```
